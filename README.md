# КиноКаталог (Movie Catalog)

Это веб-приложение для каталогизации фильмов, разработанное на React (Frontend) и Node.js/SQLite (Backend).

## Требования
Для запуска вам понадобится установленный **Node.js** (версии 16 или выше).
Скачать можно здесь: https://nodejs.org/

## Установка и запуск

1. **Распакуйте архив** в удобную папку.
2. Откройте эту папку в терминале (или командной строке).
3. Установите зависимости одной командой:
   ```bash
   npm install
   ```

4. Запустите проект (и сервер, и клиент) одной командой:
   ```bash
   npm run dev:all
   ```

5. Приложение откроется в браузере автоматически (обычно по адресу http://localhost:5173).

## Функционал
- Просмотр списка фильмов.
- Поиск по названию.
- Добавление новых фильмов.
- Удаление фильмов.
- Добавление в избранное.
- Темная тема (Dark Mode).

## Технологии
- **Frontend:** React, Vite, TypeScript, Redux Toolkit.
- **Backend:** Node.js, Express.
- **Database:** SQLite (файл `database.sqlite` создается автоматически).
